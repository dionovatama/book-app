# Fullstack Tasks (Junior)

> **Level**: Junior (0–2 years experience)

### 🔹 Fullstack Developer

Use the `frontend/` and `backend/` from this repo as your starting point.

- **Required:**
  1. **Search** — Implement search by book title and/or author on the Library > Browse page (real-time as user types)
  2. **Category filter** — Implement filter by genre/category; filter should work alongside search
  3. **Detail page** — Add a book detail page (e.g. `/books/:id`) showing: cover, title, author, genre, pages, rating, and an add-to-library button
  4. **Backend route** — Add `GET /api/books/:id` endpoint to serve a single book for the detail page

- **Optional (implement at least one):**
  - Loading and empty states (spinner while fetching, "No results found" message)
  - Error handling when the API is unavailable (show a friendly message)
  - Preserve search/filter state when navigating back from the detail page

- **Bonus:**
  - Add more essential features you think the app is missing
